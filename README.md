# PatriotHack
This is a website to track your fitness journey.
Our program is named TrackYourFitness, where the program takes user’s health information and assists users in defining what path is the best to their fitness journey.

## Installation

1. Install VS Code:

· Our program uses VS Code to run. If you don’t already have your VS Code downloaded, follow this direction https://code.visualstudio.com/

2. Install Live Server Extension

· Open VS Code

· Go to Extensions (on the left sidebar)

· Search for “Live Server”

· Download the application.

3. File Download

· Download the ZIP folder for this project

· Extract the folder to whatever directory of your choice

## Usage (Recommended with Live Server)

1. Start opening the folder in VS Code

2. Locate the file “homepage.html”

3. Right click the file and click “Open with Live Server”

4. The action will automatically open the program in your preferred searching app

5. Hit “Yes” when you’re ready to change your life!

6. Input your information and see where it goes from there

Or

1. Download the Zip folder

2. Extract the compressed folder

3. Locate the file “homepage.html”

4. Make sure to open it with “Chrome”, otherwise the image won’t render properly.
